# 水管图片说明

这个文件夹用于存放水管游戏所需的图片资源。

## 需要的图片文件

只需要提供 **4张基础图片**，程序会自动旋转这些图片来显示不同方向的水管：

### 1. pipe_end.png - 单开口（终点）
- **基础方向**：开口向上（↑）
- 程序会自动旋转到 0°、90°、180°、270° 来显示四个方向

### 2. pipe_straight.png - 直线管道
- **基础方向**：水平直线（←→）
- 程序会自动旋转到 0°、90° 来显示水平和垂直方向

### 3. pipe_corner.png - 弯角管道
- **基础方向**：右下弯角（└）
- 程序会自动旋转到 0°、90°、180°、270° 来显示四个方向的弯角

### 4. pipe_t.png - T型管道
- **基础方向**：缺失上方向（即开口为右、下、左，形状如 ┬）
- 程序会自动旋转到 0°、90°、180°、270° 来显示四个方向的T型

## 图片要求

- **格式**：PNG（推荐）或 JPG
- **尺寸**：建议 60x60 像素或更大（程序会自动缩放到 56x56 像素）
- **背景**：透明或白色
- **风格**：建议使用像素风格或卡通风格

## 注意事项

- 如果某个图片文件不存在，程序会自动使用文字符号代替
- 图片会自动缩放到 56x56 像素（留出2像素边距）
- 程序使用缓存机制，旋转后的图片会被缓存，避免重复旋转
- 可以随时替换图片，程序会在下次运行时自动加载新图片

## 基础方向示意图

```
pipe_end.png:      pipe_straight.png:   pipe_corner.png:     pipe_t.png:
    ↑                  ←→                    └                  ┬
```
